projects

These are my personal projects:

This website!

Made from scratch by yours truly, using HTML, CSS3, JavaScript, jQuery, D3.js, and a bit of Bootstrap. The little line animation in the home tab was made by randomly generating SVG paths using JavaScript and animating them with CSS3. I toyed around with a few other ideas, namely drawing specific objects, or generating ASCII images in random orders, but I eventually decided I liked the look of this the most!

"tattle.tech" Online Harassment Report Tool Chrome Extension (Winner of the #HackHarrassment "Best Anti-Bullying Hack" at HackRiddle 2016)

The Chrome extension portion of the tattle.tech tool made by some friends and I at HackRiddle 2016. It allows you to submit an abuse report to a local authority monitoring any tattle.tech reports. Includes a screenshot of the page and the URL of the reported abuse, along with a categorization and additional details about the reporter and the incident itself. This part was made entirely by me and can be found on my GitHub here.

"Peter O'Tool" Chrome extenstion (Early development)

In the very early stages still, I plan to develop a simple extension for use with Google Chrome that would bring up a larger version of an image on hover, but will also display an image of Peter O'Toole at random on occasion, with the frequency of this occurence being adjustable. A novelty extension, to be sure, but it will mostly serve as a learning tool (and probably as a funny story at some point).
